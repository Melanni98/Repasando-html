Semanticas Estructurales

semanticas estructurales
DIV: Es una etiqueta contenedora que no tiene semantica
HEADER: cabecera de un sitio web o una seccion
MAIN: define la seccion principal del documento, solo puede existir una por documento
FOOTER: pie de pagina de un sitio web o una seccion
NAV: representa un menú de navegación
ARTICLE: representa una seccion de autocontenido (que por si solo se explica)
ASIDE: representa contenido complementario o secundario
SECTION: representa una seccion de contenido genérico
ADDRESS: representa una dirección física
